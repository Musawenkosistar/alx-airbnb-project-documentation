# Airbnb Clone Backend User Stories

## Guest
1. As a Guest, I want to register and log in to my account so that I can book properties and manage my reservations.
2. As a Guest, I want to search for available properties and book them so that I can plan my stay efficiently.
3. As a Guest, I want to leave reviews for properties I stayed at so that other users can benefit from my experience.

## Host
4. As a Host, I want to add, update, and delete my property listings so that I can manage what guests see and book.
5. As a Host, I want to view all bookings for my properties so that I can prepare for upcoming reservations.

## Admin
6. As an Admin, I want to manage users and properties so that I can maintain platform integrity and security.

## Payments
7. As a Guest, I want to securely make payments for my bookings so that I can confirm reservations without issues.
